카드뒤집기
=========================================
MemoryCardGame : 

size = 4 : 카드 배열 (4*4)
seconds = 3 : 뒤집은 카드가 다른카드일때(틀렸을떄) 3초동안만 보이게하고 다시 숨김
String[][] board = new String[size][size] : 카드 저장용 배열
boolean[][] revealed = new boolean[size][size] : 카드 공개여부 배열(true 값이면 카드 앞면이 보이게, false값이면 카드 뒷면이 보이게)

****
initializeBoard() : 

String[] list에 저장된 카드(이모지) 배열을 cards라는 String형 List에 저장
Collections.shuffle(cards) : 카드(이모지) 순서 섞는 라이브러리
for문 : 카드 저장용 4*4 배열(board)에  순서 섞은 이모지 배열을 저장하고, 카드 공개여부 배열에 모두 false값(안보이게) 저장

****
printBoard(): 현재 보드판을 출력해주는 메서드

for문으로 돌리면서 revealed[i][j]값이 true면 카드 앞면(이모지),  false면 카드 뒷면( * ) 표시함.

****
isValidMove(int x1, int y1, int x2, int y2) : 사용자에게 뒤집을 카드를 입력받았을 때 알맞은 입력값인지 확인하는 메서드

알맞은 입력값
( 
   - 보드판 안의(0~3) 값이고
   - 이미 뒤집혀져 있는(!revealed) 카드가 아님,
   - 입력받은 두 카드가 같은 카드( 같은 값 )가 아님 
)
이면  true값, 아니면 false값을 return한다.

****
isGameOver() : 게임이 끝났는지(모든 카드가 다 뒤집어 졌는지) 확인하는 메서드

for문 : revealed[i][j]값이 모두 true면 true값 반환함
true값 반환시 메인함수의 while(!isGameOver()) ..탈출? 게임종료 메시지 출력

****
pause() : 설정 시간동안 멈춤

종료시간 = 시작시간 + 설정시간 
종료시간이 지나면 끝

****
clearConsole() : 콘솔창 위로 올려보내기 위한 메서드

=========================================

카드뒤집기게임( 메인함수 ) :

int computerTurn = 0
int userTurn = 0
점수 저장용 변수 (카드 뒤집기 성공시 +1 됨)

****
사용자 차례 :

뒤집을 카드 입력받고
오류처리: 숫자가 아닌 값 입력시 예외처리, '숫자만 입력 가능합니다.' 출력
sc.nextLine() : 버퍼 지우기 위해 사용
( 지우지 않으면 예외처리했던 문자가 계속 버퍼에 남아있기 때문에 무한루프에 빠질수있음 )

입력받은 카드가 유효한 값이면 revealed 배열 값을 true값으로 바꿔서 뒤집고 보드를 보여준다.
유효하지 않은 값이면 '다시 시도하세요' 메시지 출력 후 다시 뒤집을 카드 입력받음

두 카드가 같은 카드가 아니면 revealed 배열 값을 다시 false값으로 바꿔주고 
콘솔창을 위로 올려서 이전 판을 안보이게 함.

두 카드가 같은 카드면  '사용자가 카드를 맞췄습니다' 메시지 출력, userTurn+1

****
컴퓨터 차례 : 

if (!game.isGameOver()) : 사용자 차례에서 모든 카드가 뒤집어졌을때 컴퓨터 차례로 넘어가지 않고 while문을 나간다

랜덤으로 카드 선택
선택한 두 카드가 같으면 
'컴퓨터가 카드를 맞췄습니다' 메시지 출력, computerTurn+1

****
게임이 끝나면

'모든 카드를 뒤집었습니다'
사용자 점수, 컴퓨터 점수 비교해서
'사용자 승리' or '컴퓨터 승리' 출력
